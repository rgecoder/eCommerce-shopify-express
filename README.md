## FEATURES :tada:
- [x] Authentication with OAuth
- [x] Live Search
- [x] Cart
- [x] Checkout Item Changing
- [x] Instant Buy
- [x] Contact us form
- [x] Categories With Subcategories
- [x] Dynamic Content
- [x] Newsletter
- [x] Admin Panel


## TECHNOLOGY :computer:
- NodeJS - Server
- ExpressJS - Nodejs framework
- MongoDB - Database
- HTML, CSS, JS, EJS - Frontend
- PM2 - Clustering
- Firebase - Firestore for Session Storage
